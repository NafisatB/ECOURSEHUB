# Assignment:
1. What is NPM & package.json?
NPM (Node Package Manager) is a tool that allows developers to install, manage, and share JavaScript packages or libraries.
package.json is a file that stores metadata about your project, including dependencies, scripts, project name, version, and other configuration needed to run and maintain your project.

2. What is Vite?
Vite is a fast development build tool for modern web projects. It provides fast server start and instant hot module replacement for frontend development.

3. Project Setup
This project was set up using Vite with the React + JavaScript template:

bash npm create vite@latest my-app -- --template react cd my-app npm install npm run dev
